如何配置 gitconfig 連線多個 GitHub 帳號 (window)

2022年4月15日 · 4 分钟阅读
Callum
Front End Engineer

建立 SSH​

開啟終端機(快捷鍵 win+q 輸入 cmd)，接著按照步驟往下走

ssh-keygen

系統會詢問 ssh 存放位置 (建議集中擺放 .ssh 資料夾, 檔名則設置 id_rsa_username)

  • 如果不需要替換則按 enter
  • 需要替換請輸入完整路徑包含 id_rsa 是檔名
Generating public/private rsa key pair.
Enter file in which to save the key (/home/username/.ssh/id_rsa):

ssh 是否要設定密碼，如不需要請按 enter

Enter passphrase (empty for no passphrase):
Enter same passphrase again:

接著去 ssh 存放位置查看 ssh 已產生了沒 ?

  • id_rsa (私鑰)
  • id_rsa_pub(公鑰)

公鑰是要複製內容放在伺服器驗證的，私鑰則是自己保存。 當把資料推上伺服器時會以私鑰驗證公鑰。

設定 config​

請先根據你的帳號數，建立對應數量的 SSH 再往下閱讀

  1. 前往 .ssh 資料夾新增 config 檔案 (建立 text 再連同副檔名 一併刪掉改成 config)
  2. 依下方填寫內容，把 username1 換成你的 username
#username1 account
Host github.com-username1
    HostName github.com
    User git
    IdentityFile ~/.ssh/id_rsa_username1

#username2 account
Host github.com-username2
    HostName github.com
    User git
    IdentityFile ~/.ssh/id_rsa_username2

設定 gitconfig​

  1. 接著來到你的使用者路徑底下，找全域 gitconfig 檔案位置
  2. 於資料夾內建立 gitconfig-username1, gitconfig-username2 分別設定 user, email
# gitconfig-username1
[user]
    name = username1
    email = username1@gmail.com
# gitconfig-username2
[user]
    name = username2
    email = username2@gmail.com

接著打開 gitconfig 加入以下設定

[includeIf "gitdir/i:E:/Documents/Code/username1/"]
    path = ~/.gitconfig-username1
[includeIf "gitdir/i:E:/Documents/Code/username2/"]
    path = ~/.gitconfig-username2

上面的意思

  • 當 git 的儲存庫在 username1 資料夾底下時引入 ~/.gitconfig-username1 檔案
  • username2 就以此類推...

測試​

上述步驟完成後我們開始測試一下環境，先切換到 username1 底下

cd E:/Documents/Code/username1

建立 project 並且在 project 資料夾內 git init (初始化)

PS E:\Documents\Code\username1\project> git init

接著輸入 git config user.name

PS E:\Documents\Code\username1\project> git config user.name

回傳 username1 就表示設定成功囉，接下來試著使用 ssh 克隆 GitHub 儲存庫

下面是你在 GitHub ssh 複製取到的字串

git@github.com:username1/example.git

請改成

git@github.com-username1:username1/example.git

因為請往上回看 config 我們已經幫 github.com 分流

  • github.com-username1
  • github.com-username2

接下來就可以輸入下面指令

git clone git@github.com-username1:username1/example.git

結束這樣就成功做好區分兩個 git 本地 user 兩個 github 帳戶

總結​

當你身分是 username1 的時候進行開發，請把 git 儲存庫建立在 E:\Documents\Code\username1\ 底下

當你身分是 username2 的時候進行開發，請把 git 儲存庫建立在 E:\Documents\Code\username2\ 底下
