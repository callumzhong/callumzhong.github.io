Mock API 身為前端該了解的知識點​

簡介​

Mock 詞意為模擬的意思，那麼先了解為什麼需要模擬?

我們所開發的網頁應用程式大部分搭配資料庫，較大型的網頁則可能會有多個複雜的資料庫， 這需要後端開發人員的協作。

但是開發需要時間去完成，為了讓整體開發流程順利，則模擬 API 回傳數據會是好的解決方案。另外網頁有使用第三方 API 的話，利用 MOCK API 於開發期間也能降低調用第三方 API 的成本。
