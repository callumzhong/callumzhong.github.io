開始到某一天的 React 系列

May 22, 2022 · 2 min read
Callum
Front End Engineer

前言​

隨著新版 React Docs 的更新進度 => Learn React: ~70% finished.
是時候重新整理知識 🤔

系列是開始到某一天結束 (放棄) 文章更新等於知識更新 ~ 🎉

💡 小知識 :
詳細了解 React Docs 進度請至以下 issues 查看
https://github.com/reactjs/reactjs.org/issues/3308

目錄​

  1. 讓我們探討本質
  2. 什麼是 JSX 語法
  3. 建立元件與嵌套元件 (fragments)
  4. 透過 props 傳遞數據
  5. 加入元件樣式
  6. 實作靜態頁面練習拆分元件層級
  7. DOM 事件處理
  8. 單一狀態管理(useState)
  9. 實作表單練習事件處理與狀態管理
  10. 受控元件與非受控元件
  11. 條件與列表
  12. 實作 Todolist 練習列表與複習先前技能點
  13. DOM 實體取得與傳遞 (useRefs, forward, useImperativeHandle)
  14. 生命週期 (useEffect, HTTP Method )
  15. 多狀態依賴管理(useReducer)
  16. 全域狀態管理 (useContext)
  17. 優化性能 (useMemo, useCallback)
  18. Hooks 規則與自定義
  19. 入門 Redux 狀態管理工具
  20. 使用 react-router 建立多頁面
  21. 部屬 Web 應用程式
  22. 元件動起來 (transition)
