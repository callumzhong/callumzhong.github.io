Toast 2 - 快速導入 Bootstrap 5 進行開發

May 16, 2022 · 5 min read
Callum
Front End Engineer

前言​

隨著 Bootstrap 5 的來臨
開發者該怎麼面對公司的導入需求 ?

建立專案​

  1. 新增 project 資料夾
  2. 執行 yarn init -y
  3. 安裝 Prarcel 構建工具
yarn add --dev parcel
  1. 新增以下資料夾結構 ![image.png](https://cdn.hashnode.com/res/hashnode/image/upload/v1652672167208/3fOQLZkZP.png align="left")

安裝與配置​

接下來我們開始安裝 Bootstrap 5

yarn add bootstrap@latest

安裝好後，請到以下路徑打開檔案複製內容

node_modules\bootstrap\scss\bootstrap.scss

請到 project 的 src/scss 資料夾下建立 app.scss 把內容貼上 並修改 @import 連結

主題​

我們已成功導入 Bootstrap
但是依設計稿內容，多少需要更改 Bootstrap 的主題

  1. 顏色
  2. 間距
  3. 行高
  4. 各式各樣的客製化...

所以首先導入變數
採用 Sass 7-1 pattern

![image.png](https://cdn.hashnode.com/res/hashnode/image/upload/v1652674830701/Jp7sEnPyS.png align="left")

我們新增 components, pages, utils 資料夾，後面可以視需求再增加到 7-1 規格
接著 utils 底下新增 _variables.scss 檔案，填寫以下內容

// 引入 bootstrap variables 原始資料
@import '~bootstrap/scss/variables';

// 前往 @import '~bootstrap/scss/variables'; 檔案內，拉取顏色配置並把 !default 刪掉
// scss-docs-start theme-color-variables
$primary: #ff0000; //$blue;
$secondary: $gray-600;
$success: $green;
$info: $cyan;
$warning: $yellow;
$danger: $red;
$light: $gray-100;
$dark: $gray-900;
// scss-docs-end theme-color-variables

// scss-docs-start theme-colors-map
$theme-colors: (
    'primary': $primary,
    'secondary': $secondary,
    'success': $success,
    'info': $info,
    'warning': $warning,
    'danger': $danger,
    'light': $light,
    'dark': $dark,
);
// scss-docs-end theme-colors-map

上述操作如下

  1. 引入 bootstrap variables 原始資料
  2. 拉取原始資料的顏色配置並移除 !default
  3. 改變主題色 primary 色碼為 #ff0000

接下來前往 src/scss/app.scss
修改@import variables 路徑

@import './utils/variables'; // '~bootstrap/scss/variables';

查看結果 primary 已變更顏色為 #ff0000

![image.png](https://cdn.hashnode.com/res/hashnode/image/upload/v1652675587294/qO5nN0GNh.png align="left")

接下來請依此邏輯，照著需求客製化主題即可。

自定義 utilities 工具​

在 Bootstrap 4 的時代，對於 Bootstrap 不足的 utilities
需要自己寫 scss.file 檔案引入.

但是 Bootstrap 5 已經提供快速的撰寫方式~🎉
請到 src/scss/utils 底下新增 _utilities.scss 填寫以下內容

@import '~bootstrap/scss/utilities';

$utilities: map-merge(
    $utilities,
    (
        'cursor': (
            property: cursor,
            class: cursor,
            responsive: true,
            values: auto pointer grab,
        ),
    )
);

請到 src/scss/app.scss 替換 @import utilities 路徑

@import './utils/utilities'; //'~bootstrap/scss/utilities';

請到 src/index.html 新增下面內容做測試

<p class="cursor-pointer">pointer</p>
